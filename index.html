HTML

  • Structure and content
  • The head element contains information about webpage
  • The body element represents the visible content shown to the user

CSS

  • Style and presentation of a webpage.
  • Inline CSS: Quick permanent changes. Must be edited individually.
  • Internal CSS Style Sheet: Contains style for the whole webpage. Lives in the Head section.
  • External CSS Style Sheet: Most Common. Keeps CSS rules in separate file. Uses link file in Head
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Control system. Tracks changes made to file
  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it

JavaScript

  • Allows users to interact with website. Located in body of HTML.
  • Variables - containers that store values.
  • String - a named variable (In quotes)
  • Number - a numeric variable (no quotes)
  • Boolean - True/false (no quotes)
  • Array - multiple values in a single reference.
  • Assignment - "=" - assigns value
  • Strict equality - "===" - Tests to see if two values are equal and of the same data (results in boolean)
  • Conditionals - "if" "else" statements
  • Control Code - Order in which a copmuter executes code in a script (JS usually works from top to bottom unless an "if" statement is present).
  • 1st item in an array is assigned "0" not "1"
  • Function - Set of instructions that tells the computer how to perform a certain task
  • A funtions name is followed by parentheses
